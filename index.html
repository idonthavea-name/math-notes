第十二章 Sturm-Liouville 問題與特徵函數展開

Sturm-Liouville Problems & Eigenfunction Expansions

12.1 特徵值、特徵函數與 Sturm-Liouville 問題

標準 Sturm-Liouville 方程式

起始形式：

(12.1)$$y''+R(x)y'+(Q(x)+\lambda P(x))y=0$$

兩邊同乘積分因子 $r(x)=e^{\int R(x)\,dx}$ 後可改寫為標準形式：

(12.2)$$(ry')' + (q+\lambda p)\,y = 0$$
基本假設

$p,\,q,\,r$ 在區間 $(a,b)$ 或 $[a,b]$ 上連續，且 $r(x)>0$、$p(x)>0$ 於 $a < x < b$。

目標：找到使此方程式存在非平凡解 $y(x)$ 的 $\lambda$ 值。

特徵值 (eigenvalue)：使非平凡解存在的 $\lambda$。
特徵函數 (eigenfunction)：對應 $\lambda$ 的非平凡解 $y(x)$。

三種邊值問題類型

1. 正規 (Regular) S-L 問題

邊界條件：

$$A_1 y(a)+A_2 y'(a)=0,\quad B_1 y(b)+B_2 y'(b)=0$$

$A_1,A_2$ 不全為零；$B_1,B_2$ 不全為零。

2. 週期性 (Periodic) S-L 問題

條件：$r(a)=r(b)$，邊界條件：

$$y(a)=y(b),\quad y'(a)=y'(b)$$
3. 奇異 (Singular) S-L 問題

$r(a)=0$ 或 $r(b)=0$（不全為零），僅在 $r\neq 0$ 的端點設定邊界條件。

若 $r(a)=0$：僅有 $B_1 y(b)+B_2 y'(b)=0$
若 $r(b)=0$：僅有 $A_1 y(a)+A_2 y'(a)=0$

定理 12.1 ─ S-L 問題的一般性結論
  1. 正規與週期性 S-L 問題具有無限多個相異特徵值 $\lambda_1,\lambda_2,\dots$，且 $\displaystyle\lim_{n\to\infty}\lambda_n=\infty$。
  2. 所有 S-L 問題的特徵值均為實數。
  3. 特徵函數的任何非零常數倍仍為特徵函數。
  4. 正規問題中，同一特徵值的兩個特徵函數必為彼此之常數倍（唯一性，up to scalar）。
範例 12.1 ─ 正規問題 $y''+\lambda y=0;\;y(0)=y(L)=0$

區間 $[0,L]$，正規問題。

Case 1：$\lambda = 0$

$y=cx+d$，代入 BC 得 $c=d=0$ → 平凡解。$\lambda=0$ 不是特徵值。

Case 2：$\lambda < 0$（令 $\lambda=-k^2,\;k>0$）

$y=c_1 e^{kx}+c_2 e^{-kx}$。由 $y(0)=0$ 得 $c_2=-c_1$。

$y(L)=c_1(e^{kL}-e^{-kL})=0$，但 $e^{kL}\neq e^{-kL}$ → $c_1=0$。無負特徵值。

Case 3：$\lambda > 0$（令 $\lambda=k^2,\;k>0$）

$y=c_1\cos(kx)+c_2\sin(kx)$。$y(0)=c_1=0$，故 $y=c_2\sin(kx)$。

$y(L)=c_2\sin(kL)=0$。取 $c_2\neq 0$ → $\sin(kL)=0$ → $kL=n\pi$。

$$\boxed{\lambda_n = \left(\frac{n\pi}{L}\right)^2,\quad \varphi_n(x)=\sin\!\left(\frac{n\pi x}{L}\right),\quad n=1,2,3,\dots}$$
範例 12.2 ─ 週期性問題 $y''+\lambda y=0;\;y(-L)=y(L),\;y'(-L)=y'(L)$

區間 $[-L,L]$，$r(x)=1$ 故 $r(-L)=r(L)$。

Case 1：$\lambda=0$

$y=cx+d$。由 BC 得 $c=0$。$\lambda=0$ 是特徵值，特徵函數為常數 $\varphi_0=1$。

Case 2：$\lambda<0$

$y=c_1 e^{kx}+c_2 e^{-kx}$。聯立 BC 得 $c_1=c_2$ 再得 $c_1=0$。無負特徵值。

Case 3：$\lambda>0$

$y=c_1\cos(kx)+c_2\sin(kx)$。BC 要求 $\sin(kL)=0$ → $kL=n\pi$。

$$\boxed{\lambda_n = \left(\frac{n\pi}{L}\right)^2,\quad \varphi_n(x)=c_1\cos\!\left(\frac{n\pi x}{L}\right)+c_2\sin\!\left(\frac{n\pi x}{L}\right),\quad n=0,1,2,\dots}$$
注意

週期性問題中，每個正特徵值可對應兩個獨立特徵函數：$\cos(n\pi x/L)$ 與 $\sin(n\pi x/L)$。令 $n=0$ 可整合 Case 1 與 Case 3。

範例 12.3 ─ Neumann BC：$y''+\lambda y=0;\;y'(0)=y'(L)=0$

正規問題，區間 $[0,L]$。$\lambda=0$ 是特徵值（$\varphi_0=1$）。無負特徵值。

$$\boxed{\lambda_n = \frac{n^2\pi^2}{L^2},\quad \varphi_n(x)=\cos\!\left(\frac{n\pi x}{L}\right),\quad n=0,1,2,\dots}$$
範例 12.4 ─ 輻射邊界：$y''+\lambda y=0;\;y(0)=0,\;y'(L)+Ay(L)=0$

$A > 0$ 為正數常數（與能量散失率有關）。無負特徵值， $\lambda = 0$ 亦非特徵值。

令 $\lambda=k^2$，由 BC 得：

$$\tan(kL) = -\frac{k}{A}$$

此為超越方程式，須用圖解法或數值方法求解。令 $z=kL$：

$$\tan(z)=-\frac{z}{AL}$$

直線 $w=-z/(AL)$ 與 $w=\tan(z)$ 在 $z>0$ 有無限多交點 → 無限多特徵值。

特徵函數的完備性 (Completeness)

完備性定義

如果一組特徵函數的列表中，任何其他特徵函數都能表示為列表中特徵函數的線性組合或常數倍數，則該列表是完備的。

不完備的列表會遺漏無法從中還原的資訊。

12.2 特徵函數展開 (Eigenfunction Expansions)

核心想法：將函數 $f(x)$ 寫成特徵函數的無窮級數：

(12.3)$$f(x)=\sum_{n=1}^{\infty}c_n\,\varphi_n(x)$$

定理 12.2 ─ 特徵函數的加權正交性

定理 12.2

對應於相異特徵值的特徵函數，在 $[a,b]$ 上以權重函數 $p(x)$ 正交：

(12.4) $$\lambda\neq\mu \implies \int_a^b p(x)\,\varphi(x)\,\psi(x)\,dx = 0$$

其中 $p(x)$ 即標準 S-L 方程式 (12.2) 中 $\lambda$ 的係數。

定理 12.2 證明（展開 / 收合）

已知 $\varphi$ 對應 $\lambda$，$\psi$ 對應 $\mu$：

$$(r\varphi')' + (q+\lambda p)\varphi = 0, \quad (r\psi')' + (q+\mu p)\psi = 0$$

第一式乘 $\psi$、第二式乘 $\varphi$，相減得：

$$(r\psi')'\varphi - (r\varphi')'\psi + (\lambda-\mu)\,p\,\varphi\,\psi = 0$$

定義 Wronskian $W = \varphi\psi' - \psi\varphi'$，可驗證：

$$\frac{d}{dx}(rW) = (\lambda-\mu)\,p\,\varphi\,\psi$$

兩邊從 $a$ 積到 $b$：

(12.5) $$(\lambda-\mu)\int_a^b p\,\varphi\,\psi\,dx = r(b)W(b) - r(a)W(a)$$

由邊界條件可證 $W(a)=W(b)=0$（正規情況下對兩端分別操作），因此右側為零。由於 $\lambda\neq\mu$，得 $\displaystyle\int_a^b p\,\varphi\,\psi\,dx=0$。

展開係數公式

在 $f(x)=\sum c_k\varphi_k(x)$ 兩邊乘 $p(x)\varphi_n(x)$ 再積分，利用正交性消去 $k\neq n$ 的項：

核心公式
(12.6) $$\boxed{c_n = \frac{\displaystyle\int_a^b p(x)\,f(x)\,\varphi_n(x)\,dx}{\displaystyle\int_a^b p(x)\,\varphi_n^2(x)\,dx}}$$
定理 12.3 ─ 特徵函數展開的收斂性

設 $\varphi_n$ 為完備特徵函數、$c_n$ 由 (12.6) 給出、$f(x)$ 在 $[a,b]$ 上分段平滑。

則對 $a < x < b$，級數 $\sum c_n \varphi_n(x)$ 收斂至：

$$\frac{1}{2}\bigl[f(x^-)+f(x^+)\bigr]$$

連續點 → 收斂至 $f(x)$。跳躍不連續點 → 收斂至左右極限的平均值。

範例 12.5

$y''+\lambda y=0;\;y'(0)=y'(2\pi)=0$，$L=2\pi$。

特徵值 $\lambda_n = n^2/4$，特徵函數 $\varphi_n(x)=\cos(nx/2)$，$n=0,1,2,\dots$。權重 $p(x)=1$。

展開 $f(x)=\begin{cases}x & 0\le x<2 \\ 6 & 2\le x\le 2\pi\end{cases}$

完整推導

$c_0$：

$$c_0 = \frac{\int_0^{2\pi}f(x)\cdot 1\,dx}{\int_0^{2\pi}1^2\,dx} = \frac{\int_0^2 x\,dx + \int_2^{2\pi}6\,dx}{2\pi} = \frac{2 + 6(2\pi-2)}{2\pi} = \frac{12\pi - 10}{2\pi} = 6 - \frac{5}{\pi}$$

$c_n$（$n \ge 1$）：

$$c_n = \frac{\int_0^{2\pi}f(x)\cos(nx/2)\,dx}{\int_0^{2\pi}\cos^2(nx/2)\,dx}$$

分母：$\int_0^{2\pi}\cos^2(nx/2)\,dx = \pi$

分子拆兩段：

$$\int_0^2 x\cos(nx/2)\,dx = \left[\frac{2x}{n}\sin(nx/2)\right]_0^2 + \frac{4}{n^2}[\cos(nx/2)]_0^2 = \frac{4}{n}\sin(n) + \frac{4}{n^2}(\cos n - 1)$$ $$\int_2^{2\pi}6\cos(nx/2)\,dx = \frac{12}{n}[\sin(nx/2)]_2^{2\pi} = \frac{12}{n}(\sin(n\pi) - \sin(n)) = -\frac{12}{n}\sin(n)$$

合計分子 $= \frac{4}{n}\sin n + \frac{4}{n^2}(\cos n - 1) - \frac{12}{n}\sin n = \frac{4(\cos n -1)}{n^2} - \frac{8\sin n}{n}$

$$\boxed{c_n = \frac{4(\cos n - 1)}{n^2\pi} - \frac{8\sin n}{n\pi}}$$

收斂：連續處收斂至 $f(x)$。在 $x=2$（跳躍處），收斂至 $\frac{1}{2}(2+6)=4$。

範例 12.6 ─ 非標準形式

$y''+4y'+(3+\lambda)y=0;\;y(0)=y(1)=0$，展開 $f(x)=x$。

完整推導

Step 1：求特徵值

令 $y=e^{rx}$ → $r^2+4r+(3+\lambda)=0$ → $r = -2\pm\sqrt{1-\lambda}$。

$1-\lambda=0$：重根 $r=-2$，$y=(c_1+c_2 x)e^{-2x}$，BC 強制 $c_1=c_2=0$。$\lambda=1$ 不是特徵值。

$1-\lambda > 0$：$r=-2\pm k$，$y=c_1 e^{(-2+k)x}+c_2 e^{(-2-k)x}$，BC 同樣只有平凡解。

$1-\lambda < 0$：令 $1-\lambda = -k^2$（$k>0$），$r=-2\pm ik$。

$$y = e^{-2x}(c_1\cos(kx)+c_2\sin(kx))$$

$y(0)=c_1=0$ → $y=c_2 e^{-2x}\sin(kx)$。$y(1)=c_2 e^{-2}\sin(k)=0$ → $k=n\pi$。

$$\lambda_n = 1+n^2\pi^2, \quad \varphi_n(x)=e^{-2x}\sin(n\pi x)$$

Step 2：化為標準形式求 $p(x)$

$y'' + 4y' + (3+\lambda)y = 0$ 中 $R(x)=4$，乘以 $e^{\int 4\,dx}=e^{4x}$：

$$(e^{4x}y')' + (3e^{4x} + \lambda e^{4x})y = 0$$

比較標準形式 → $p(x) = e^{4x}$。

Step 3：求係數

$$c_n = \frac{\int_0^1 e^{4x}\cdot x\cdot e^{-2x}\sin(n\pi x)\,dx}{\int_0^1 e^{4x}\cdot e^{-4x}\sin^2(n\pi x)\,dx} = \frac{\int_0^1 x\,e^{2x}\sin(n\pi x)\,dx}{\int_0^1 \sin^2(n\pi x)\,dx}$$

分母：$\int_0^1 \sin^2(n\pi x)\,dx = 1/2$

分子：用兩次分部積分，令 $I = \int_0^1 x\,e^{2x}\sin(n\pi x)\,dx$

$$I = \left[\frac{x\,e^{2x}(2\sin(n\pi x)-n\pi\cos(n\pi x))}{4+n^2\pi^2}\right]_0^1 - \int_0^1 \frac{e^{2x}(2\sin(n\pi x)-n\pi\cos(n\pi x))}{4+n^2\pi^2}\,dx$$

邊界項：$x=1$ 時 $\sin(n\pi)=0$，$\cos(n\pi)=(-1)^n$，故為 $\frac{-n\pi(-1)^n e^2}{4+n^2\pi^2}$。

剩餘積分再用公式 $\int e^{2x}\sin(n\pi x)\,dx$ 和 $\int e^{2x}\cos(n\pi x)\,dx$，最終：

$$\boxed{c_n = \frac{2I}{1} = \frac{-8n\pi - 2e^2 n^3\pi^3(-1)^n}{(4+n^2\pi^2)^2}}$$

12.2.1 係數的性質

正規化特徵函數 (Normalized Eigenfunctions)：

$$\Phi_n(x) = \frac{\varphi_n(x)}{\sqrt{\int_a^b p(x)\,\varphi_n^2(x)\,dx}}$$

使得 $\int_a^b p(x)\,\Phi_n^2(x)\,dx = 1$，係數簡化為：

(12.8)$$c_n = \int_a^b p(x)\,f(x)\,\Phi_n(x)\,dx$$

最佳最小平方逼近：選 $d_n=c_n$ 使得 $\int_a^b p(x)\bigl(f(x)-\sum d_n\Phi_n\bigr)^2 dx$ 最小。

Bessel 不等式 & Parseval 等式
(12.12) $$\sum_{n=1}^{\infty}c_n^2 \le \int_a^b p(x)\bigl(f(x)\bigr)^2 dx \quad\text{(Bessel 不等式)}$$

若 $f$ 連續且 $f'$ 分段連續，則等號成立 → Parseval 等式。

12.3 傅立葉級數 (Fourier Series)

傅立葉級數是特殊的特徵函數展開，應用最廣。三種常見形式：

12.3.1 $[-L,L]$ 上的傅立葉級數

來自週期性 S-L 問題（範例 12.2），完備特徵函數：

$$1,\;\cos(\pi x/L),\;\sin(\pi x/L),\;\cos(2\pi x/L),\;\sin(2\pi x/L),\;\dots$$

權重函數 $p(x)=1$。

傅立葉級數
(12.13) $$f(x) \sim \frac{a_0}{2}+\sum_{n=1}^{\infty}\left[a_n\cos\!\left(\frac{n\pi x}{L}\right)+b_n\sin\!\left(\frac{n\pi x}{L}\right)\right]$$
(12.14) $$a_n = \frac{1}{L}\int_{-L}^{L}f(x)\cos\!\left(\frac{n\pi x}{L}\right)dx,\quad n=0,1,2,\dots$$
(12.15) $$b_n = \frac{1}{L}\int_{-L}^{L}f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx,\quad n=1,2,\dots$$
定理 12.4 ─ 傅立葉級數的收斂性

若 $f(x)$ 在 $[-L,L]$ 上分段平滑，則：

• 在 $-L < x < L$：收斂至 $\dfrac{1}{2}\bigl(f(x^-)+f(x^+)\bigr)$

• 在端點 $x=\pm L$：收斂至 $\dfrac{1}{2}\bigl(f(-L^+)+f(L^-)\bigr)$

範例 12.7

$f(x)=\begin{cases}x & -3\le x<1 \\ -2+x & 1\le x\le 3\end{cases}$，$[-3,3]$ 上展開。$L=3$。

完整推導

$a_0$：

$$a_0 = \frac{1}{3}\int_{-3}^{3}f(x)\,dx = \frac{1}{3}\left[\int_{-3}^{1}x\,dx + \int_1^3(x-2)\,dx\right] = \frac{1}{3}\left[\frac{x^2}{2}\Big|_{-3}^1 + \frac{(x-2)^2}{2}\Big|_1^3\right] = \frac{1}{3}\left[-4+0\right] = -\frac{4}{3}$$

$a_n$：

$$a_n = \frac{1}{3}\int_{-3}^{3}f(x)\cos\!\left(\frac{n\pi x}{3}\right)dx = \frac{1}{3}\left[\int_{-3}^{1}x\cos\!\left(\frac{n\pi x}{3}\right)dx + \int_1^3(x-2)\cos\!\left(\frac{n\pi x}{3}\right)dx\right]$$

各段用分部積分 $\int u\cos(v)\,dx$，經計算後：

$$a_n = \frac{2}{n\pi}\sin\!\left(\frac{n\pi}{3}\right)$$

$b_n$：

$$b_n = \frac{1}{3}\int_{-3}^{3}f(x)\sin\!\left(\frac{n\pi x}{3}\right)dx = \frac{1}{3}\left[\int_{-3}^{1}x\sin\!\left(\frac{n\pi x}{3}\right)dx + \int_1^3(x-2)\sin\!\left(\frac{n\pi x}{3}\right)dx\right]$$

各段用分部積分，經計算後：

$$b_n = -\frac{2}{n\pi}\left(2(-1)^n + \cos\!\left(\frac{n\pi}{3}\right)\right)$$

收斂判斷：

$f(1^-)=1$，$f(1^+)=-2+1=-1$ → $x=1$ 跳躍，收斂至 $\frac{1}{2}(1+(-1))=0$。

$f(-3^+)=-3$，$f(3^-)=3-2=1$ → 端點收斂至 $\frac{1}{2}(-3+1)=-1$。

12.3.2 $[0,L]$ 上的傅立葉餘弦級數

來自 S-L 問題 $y''+\lambda y=0;\;y'(0)=y'(L)=0$（範例 12.3）。

餘弦級數
(12.16) $$g(x) \sim \frac{a_0}{2}+\sum_{n=1}^{\infty}a_n\cos\!\left(\frac{n\pi x}{L}\right)$$
(12.17) $$a_n = \frac{2}{L}\int_0^L g(x)\cos\!\left(\frac{n\pi x}{L}\right)dx,\quad n=0,1,2,\dots$$
定理 12.5 ─ 餘弦級數收斂性

$g$ 分段平滑。 在 $0 < x < L$ 收斂至 $\frac{1}{2}(g(x^-)+g(x^+))$。

$x=0$ 處收斂至 $g(0^+)$； $x=L$ 處收斂至 $g(L^-)$。

範例 12.8 ─ $g(x)=e^x$ 在 $[0,1]$ 上的餘弦展開
完整推導

$L=1$，餘弦係數公式 $a_n = 2\int_0^1 e^x\cos(n\pi x)\,dx$。

$a_0$： $a_0 = 2\int_0^1 e^x\,dx = 2(e-1)$

$a_n$（$n \ge 1$）：用公式 $\int e^{ax}\cos(bx)\,dx = \frac{e^{ax}(a\cos(bx)+b\sin(bx))}{a^2+b^2}$，取 $a=1$，$b=n\pi$：

$$a_n = 2\left[\frac{e^x(\cos(n\pi x)+n\pi\sin(n\pi x))}{1+n^2\pi^2}\right]_0^1 = \frac{2}{1+n^2\pi^2}\left[e\cos(n\pi) - 1\right] = \frac{2(e(-1)^n-1)}{1+n^2\pi^2}$$
$$\boxed{e^x \sim (e-1) + \sum_{n=1}^{\infty}\frac{2(e(-1)^n-1)}{1+n^2\pi^2}\cos(n\pi x)}$$

13.3 傅立葉級數的積分與微分

核心觀念

逐項積分幾乎永遠可以做（條件超寬鬆）。逐項微分經常出事（條件嚴格）。這個不對稱性是考試重點。

為什麼逐項微分危險？

範例 13.6 ─ 逐項微分的災難

$f(x) = x$ 在 $[-\pi,\pi]$ 上的傅立葉級數：

$$x = \sum_{n=1}^{\infty}\frac{2(-1)^{n+1}}{n}\sin(nx), \quad -\pi < x < \pi$$

如果天真地逐項微分（對每一項取導數）：

$$\frac{d}{dx}\left[\frac{2(-1)^{n+1}}{n}\sin(nx)\right] = 2(-1)^{n+1}\cos(nx)$$

得到級數 $\sum_{n=1}^{\infty}2(-1)^{n+1}\cos(nx)$。

但 $f'(x) = 1$，而這個級數的通項 $2(-1)^{n+1}\cos(nx)$ 不趨向零，所以級數根本不收斂，更不用說等於 $1$。

為什麼會爆？

問題出在 $f(-\pi) = -\pi \neq \pi = f(\pi)$。左右端點的值不同，傅立葉級數在端點收斂至 $\frac{1}{2}(-\pi+\pi) = 0 \neq f(\pm\pi)$。

直覺上：傅立葉級數其實是在逼近一個週期延拓後的函數。$f(x)=x$ 延拓後在 $\pm\pi$ 處有跳躍，微分把這些跳躍「放大」成不收斂的行為。

逐項積分（條件寬鬆，幾乎永遠成立）

定理 13.4 ─ 逐項積分

設 $f$ 在 $[-L,L]$ 上分段連續（這是唯一條件！），傅立葉級數為

$$\frac{a_0}{2}+\sum_{n=1}^{\infty}\left[a_n\cos\!\left(\frac{n\pi x}{L}\right)+b_n\sin\!\left(\frac{n\pi x}{L}\right)\right]$$

則對任何 $x \in [-L,L]$：

$$\int_{-L}^{x}f(t)\,dt = \frac{a_0}{2}(x+L) + \frac{L}{\pi}\sum_{n=1}^{\infty}\frac{1}{n}\left[a_n\sin\!\left(\frac{n\pi x}{L}\right) - b_n\!\left(\cos\!\left(\frac{n\pi x}{L}\right)-(-1)^n\right)\right]$$

右邊就是把傅立葉級數逐項從 $-L$ 積到 $x$ 得到的結果。

重點：即使 $f$ 有跳躍不連續點（級數在那裡不收斂至 $f$），逐項積分仍然成立。

定理 13.4 證明（展開 / 收合）

證明策略：構造一個輔助函數 $F(x)$，使它是連續的，然後展開 $F(x)$ 的傅立葉級數。

Step 1：定義輔助函數

$$F(x) = \int_{-L}^{x}f(t)\,dt - \frac{a_0}{2}x$$

$F(x)$ 是連續的（因為積分是連續函數），而且：

$$F(-L) = -\frac{a_0}{2}(-L) = \frac{a_0 L}{2}$$ $$F(L) = \int_{-L}^{L}f(t)\,dt - \frac{a_0}{2}L = La_0 - \frac{a_0 L}{2} = \frac{a_0 L}{2}$$

所以 $F(-L) = F(L)$ ✓，且 $F'(x) = f(x) - \frac{a_0}{2}$（$f$ 連續處）→ $F$ 分段平滑。

Step 2：展開 $F(x)$ 的傅立葉級數

$F$ 連續且 $F(-L)=F(L)$，所以它的傅立葉級數收斂至 $F(x)$：

$$F(x) = \frac{\alpha_0}{2} + \sum_{n=1}^{\infty}\left[\alpha_n\cos\!\left(\frac{n\pi x}{L}\right) + \beta_n\sin\!\left(\frac{n\pi x}{L}\right)\right]$$

Step 3：用分部積分把 $F$ 的係數 $\alpha_n, \beta_n$ 關聯到 $f$ 的係數 $a_n, b_n$

$$\alpha_n = \frac{1}{L}\int_{-L}^{L}F(t)\cos\!\left(\frac{n\pi t}{L}\right)dt$$

分部積分（$u = F(t)$，$dv = \cos(n\pi t/L)\,dt$），邊界項因 $\sin(n\pi)=\sin(-n\pi)=0$ 而消失：

$$\alpha_n = -\frac{1}{n\pi}\int_{-L}^{L}F'(t)\sin\!\left(\frac{n\pi t}{L}\right)dt = -\frac{1}{n\pi}\int_{-L}^{L}\left(f(t)-\frac{a_0}{2}\right)\sin\!\left(\frac{n\pi t}{L}\right)dt$$

$\int \frac{a_0}{2}\sin(\cdot)\,dt = 0$（完整週期），所以：

$$\alpha_n = -\frac{L}{n\pi}b_n$$

類似地：

$$\beta_n = \frac{L}{n\pi}a_n$$

Step 4：代回並整理

把 $\alpha_n$ 和 $\beta_n$ 代入 $F(x)$ 的傅立葉級數，再用 $F(x) = \int_{-L}^{x}f(t)\,dt - \frac{a_0}{2}x$ 還原，就得到定理的結論。$\alpha_0$ 可由 $F(L)$ 的值確定。

範例 13.7 ─ 逐項積分驗證

$f(x) = x$ 在 $[-\pi,\pi]$：$f(x) = \sum_{n=1}^{\infty}\frac{2(-1)^{n+1}}{n}\sin(nx)$

完整推導

雖然逐項微分失敗了，但逐項積分完全合法（$f$ 連續 → 分段連續 ✓）：

$$\int_{-\pi}^{x}t\,dt = \frac{x^2 - \pi^2}{2} = \sum_{n=1}^{\infty}\frac{2(-1)^{n+1}}{n}\int_{-\pi}^{x}\sin(nt)\,dt$$ $$= \sum_{n=1}^{\infty}\frac{2(-1)^{n+1}}{n}\left[-\frac{\cos(nx)}{n}+\frac{\cos(n\pi)}{n}\right] = \sum_{n=1}^{\infty}\frac{2(-1)^n}{n^2}\left(\cos(nx) - (-1)^n\right)$$

可以驗證這個級數確實等於 $\frac{x^2-\pi^2}{2}$。

注意：積分後係數多了 $1/n$ 因子（$\frac{2}{n} \to \frac{2}{n^2}$），級數收斂性變更好。這就是積分比微分安全的原因。

逐項微分（條件嚴格，三個缺一不可）

定理 13.5 ─ 逐項微分

需要同時滿足三個條件：

① $f$ 在 $[-L,L]$ 上連續

② $f(-L) = f(L)$（端點值相等）

③ $f'(x)$ 在 $[-L,L]$ 上連續

則：

1. 傅立葉級數在整個 $[-L,L]$ 上收斂至 $f(x)$（含端點）。

2. 在 $f''(x)$ 存在之處，逐項微分收斂至 $f'(x)$：

$$f'(x) = \sum_{n=1}^{\infty}\frac{n\pi}{L}\left[-a_n\sin\!\left(\frac{n\pi x}{L}\right) + b_n\cos\!\left(\frac{n\pi x}{L}\right)\right]$$
三個條件的直覺理解

① $f$ 連續：如果 $f$ 有跳躍，微分在跳躍處會產生 $\delta$ 函數，一般傅立葉級數處理不了。

② $f(-L) = f(L)$：傅立葉級數本質上是在逼近一個週期函數。如果 $f(-L) \neq f(L)$，週期延拓後在 $\pm L$ 處會產生跳躍（見範例 13.6），微分就會爆掉。

③ $f'$ 連續：如果 $f$ 有尖角（像 $|x|$ 在 $x=0$），$f'$ 在那裡不連續。逐項微分在那個點可能不等於 $f'$，但在其他點仍然有效。

定理 13.5 的證明思路（展開 / 收合）

思路：直接展開 $f'(x)$ 的傅立葉級數，然後用分部積分把 $f'$ 的係數關聯到 $f$ 的係數。

Step 1：$f'$ 連續 → $f'$ 可以展開傅立葉級數。設 $f'$ 的係數為 $\alpha_n, \beta_n$。

Step 2：用分部積分關聯係數：

$$\alpha_n = \frac{1}{L}\int_{-L}^{L}f'(x)\cos\!\left(\frac{n\pi x}{L}\right)dx$$

分部積分（$u=\cos(\cdot)$，$dv=f'(x)\,dx$）：

$$= \frac{1}{L}\left[f(x)\cos\!\left(\frac{n\pi x}{L}\right)\right]_{-L}^{L} + \frac{n\pi}{L^2}\int_{-L}^{L}f(x)\sin\!\left(\frac{n\pi x}{L}\right)dx$$

邊界項：$f(L)\cos(n\pi) - f(-L)\cos(-n\pi) = 0$（因為 $f(-L)=f(L)$！這就是條件 ② 的用處）

所以 $\alpha_n = \frac{n\pi}{L}b_n$。類似地 $\beta_n = -\frac{n\pi}{L}a_n$。

Step 3：這表示 $f'$ 的傅立葉級數就是 $f$ 的傅立葉級數逐項微分的結果 ✓

三個條件分別失敗的例子

條件 ② 失敗：$f(x) = x$ 在 $[-\pi,\pi]$

$f$ 連續 ✓，$f'=1$ 連續 ✓，但 $f(-\pi)=-\pi \neq \pi=f(\pi)$ ✗

逐項微分得 $\sum 2(-1)^{n+1}\cos(nx)$ → 不收斂。

條件 ③ 失敗：$f(x) = |x|$ 在 $[-1,1]$

$f$ 連續 ✓，$f(-1)=f(1)=1$ ✓，但 $f'$ 在 $x=0$ 不連續 ✗

逐項微分得 $\frac{4}{\pi}\sum\frac{1}{2m+1}\sin((2m+1)\pi x)$ → 在 $x \neq 0$ 處仍收斂至 $f'(x)$，但定理不保證。

三個條件全滿足：$f(x) = x^2$ 在 $[-2,2]$

$f$ 連續 ✓，$f(-2)=f(2)=4$ ✓，$f'=2x$ 連續 ✓

$$x^2 = \frac{4}{3}+\frac{16}{\pi^2}\sum_{n=1}^{\infty}\frac{(-1)^n}{n^2}\cos\!\left(\frac{n\pi x}{2}\right)$$

逐項微分：

$$f'(x) = 2x = \frac{8}{\pi}\sum_{n=1}^{\infty}\frac{(-1)^{n+1}}{n}\sin\!\left(\frac{n\pi x}{2}\right) \quad \text{for } -2 < x < 2$$

這確實是 $2x$ 在 $[-2,2]$ 上的傅立葉級數 ✓

絕對與均勻收斂

定理 13.6 ─ 絕對與均勻收斂

設 $f$ 在 $[-L,L]$ 上連續，$f'$ 分段連續，且 $f(-L)=f(L)$。

則傅立葉級數在 $[-L,L]$ 上絕對且均勻收斂至 $f(x)$。

注意：這裡 $f'$ 只需要分段連續（比定理 13.5 的「連續」更弱），但結論也不同 — 這裡只談收斂性，不談逐項微分。

定理 13.6 證明概要（展開 / 收合）

設 $f$ 的傅立葉係數為 $a_n, b_n$，$f'$ 的傅立葉係數為 $\alpha_n, \beta_n$。

Step 1：由分部積分（同定理 13.5 的證明），因 $f(-L)=f(L)$：

$$\alpha_n = \frac{n\pi}{L}b_n, \quad \beta_n = -\frac{n\pi}{L}a_n$$

Step 2：利用不等式 $(a-b)^2 \ge 0$ → $2|ab| \le a^2 + b^2$，取 $a = \alpha_n$，$b = 1/n$：

$$\frac{|\alpha_n|}{n} \le \frac{1}{2}\alpha_n^2 + \frac{1}{2n^2}$$

$\beta_n$ 也有類似不等式。兩式相加：

$$|a_n| + |b_n| = \frac{L}{n\pi}(|\beta_n|+|\alpha_n|) \le \frac{L}{2\pi}(\alpha_n^2+\beta_n^2) + \frac{L}{n^2\pi}$$

Step 3：由 Bessel 不等式，$\sum(\alpha_n^2+\beta_n^2)$ 收斂。$\sum 1/n^2$ 也收斂。

因此 $\sum(|a_n|+|b_n|)$ 收斂 → 傅立葉級數絕對收斂。

Step 4：因 $|a_n\cos(\cdot)+b_n\sin(\cdot)| \le |a_n|+|b_n|$，由 Weierstrass M-test → 均勻收斂。

積分 vs 微分的本質差異

積分讓級數更好：原本係數是 $O(1/n)$，積分後變 $O(1/n^2)$，收斂更快。

微分讓級數更差：原本係數是 $O(1/n)$，微分後多乘 $n$，變 $O(1)$，可能不收斂。

這就是為什麼積分條件寬鬆（分段連續就夠），微分條件嚴格（要三個條件全滿足）。

13.5 相角形式 (Phase Angle Form)

這節在幹嘛？一句話版本

把傅立葉級數的每一項 $a_n\cos(\cdot) + b_n\sin(\cdot)$（兩個函數加起來）合併成一個 $c_n\cos(\cdot + \delta_n)$（一個函數帶相位偏移）。本質上只是三角恆等式的換寫法，數學內容完全一樣。

前置：週期函數與基本週期

週期與基本週期

週期：$f(x+p) = f(x)$ 對所有 $x$ 成立 → $p$ 是 $f$ 的一個週期。

基本週期：最小的正數週期。例如 $\cos(x)$ 有週期 $2\pi, 4\pi, 6\pi, \dots$，但基本週期是 $2\pi$。

週期 $p$ 的函數，圖形每隔 $p$ 就重複一次。知道 $[-p/2, p/2]$ 上的圖形就知道全部。

從普通傅立葉級數出發

設 $f$ 有基本週期 $p$。令 $L = p/2$，則 $f$ 在 $[-L,L] = [-p/2, p/2]$ 上的傅立葉級數是：

$$\frac{a_0}{2} + \sum_{n=1}^{\infty}\left[a_n\cos\!\left(\frac{2n\pi x}{p}\right) + b_n\sin\!\left(\frac{2n\pi x}{p}\right)\right]$$

令 $\omega_0 = \frac{2\pi}{p}$（基本角頻率），寫得更簡潔：

$$\frac{a_0}{2} + \sum_{n=1}^{\infty}\left[a_n\cos(n\omega_0 x) + b_n\sin(n\omega_0 x)\right]$$

係數：

$$a_n = \frac{2}{p}\int_{-p/2}^{p/2}f(x)\cos(n\omega_0 x)\,dx, \quad b_n = \frac{2}{p}\int_{-p/2}^{p/2}f(x)\sin(n\omega_0 x)\,dx$$
重要：積分區間可以任意選

因為 $f$ 有週期 $p$，積分可以在任何長度為 $p$ 的區間 $[\alpha, \alpha+p]$ 上計算，結果都一樣：

$$a_n = \frac{2}{p}\int_{\alpha}^{\alpha+p}f(x)\cos(n\omega_0 x)\,dx$$

這在函數只給你 $[0, p)$ 上的定義時特別有用（取 $\alpha = 0$）。

示範：慢的做法 vs 快的做法

$f(x) = x^2$，$0 \le x < 3$，週期 $p = 3$。求 $a_0$。

❌ 慢的做法：硬用標準區間 $[-3/2,\; 3/2]$

標準區間是 $[-p/2, p/2] = [-3/2, 3/2]$，但 $f(x) = x^2$ 只在 $[0,3)$ 上有定義。

在 $[-3/2, 0)$ 上需要週期延拓：$f(x) = f(x+3) = (x+3)^2$

然後拆成兩段積分：

$$a_0 = \frac{2}{3}\left[\int_{-3/2}^{0}(x+3)^2\,dx + \int_0^{3/2}x^2\,dx\right]$$

第一段：$\int_{-3/2}^{0}(x+3)^2\,dx = \left[\frac{(x+3)^3}{3}\right]_{-3/2}^{0} = \frac{27}{3} - \frac{(3/2)^3}{3} = 9 - \frac{9}{8} = \frac{63}{8}$

第二段：$\int_0^{3/2}x^2\,dx = \frac{(3/2)^3}{3} = \frac{9}{8}$

$$a_0 = \frac{2}{3}\left[\frac{63}{8}+\frac{9}{8}\right] = \frac{2}{3}\cdot 9 = 6$$

要先推導延拓函數、拆兩段、算兩個積分。

✅ 快的做法：直接用 $[0, 3)$（取 $\alpha = 0$）

$f(x) = x^2$ 在 $[0,3)$ 上就是 $x^2$，不用延拓、不用拆段：

$$a_0 = \frac{2}{3}\int_0^3 x^2\,dx = \frac{2}{3}\cdot\frac{x^3}{3}\Big|_0^3 = \frac{2}{3}\cdot 9 = 6$$

一步到位。

答案一樣都是 $6$，但快的做法省了延拓和拆段的功夫。$a_n$ 和 $b_n$ 的計算也是同理，差別更大（因為帶 $\cos$ 和 $\sin$ 的分部積分拆兩段會更痛苦）。

核心推導：$a_n\cos + b_n\sin \to c_n\cos(\cdot + \delta_n)$

目標：找 $c_n$ 和 $\delta_n$ 使得

$$a_n\cos(n\omega_0 x) + b_n\sin(n\omega_0 x) = c_n\cos(n\omega_0 x + \delta_n)$$
推導過程（展開 / 收合）

Step 1：展開右邊

用和角公式 $\cos(A+B) = \cos A\cos B - \sin A\sin B$：

$$c_n\cos(n\omega_0 x + \delta_n) = c_n\cos(n\omega_0 x)\cos(\delta_n) - c_n\sin(n\omega_0 x)\sin(\delta_n)$$

Step 2：比較兩邊

左邊 = $a_n\cos(n\omega_0 x) + b_n\sin(n\omega_0 x)$，所以：

$$c_n\cos(\delta_n) = a_n \quad \text{...(I)}$$ $$-c_n\sin(\delta_n) = b_n \quad \text{...(II)}$$

（也就是 $c_n\sin(\delta_n) = -b_n$）

Step 3：解 $c_n$

(I)² + (II)²：

$$c_n^2\cos^2(\delta_n) + c_n^2\sin^2(\delta_n) = a_n^2 + b_n^2$$ $$c_n^2 = a_n^2 + b_n^2$$
$$\boxed{c_n = \sqrt{a_n^2 + b_n^2}}$$

Step 4：解 $\delta_n$

(II) ÷ (I)：

$$\frac{c_n\sin(\delta_n)}{c_n\cos(\delta_n)} = \tan(\delta_n) = \frac{-b_n}{a_n}$$
$$\boxed{\delta_n = -\arctan\!\left(\frac{b_n}{a_n}\right)} \quad \text{（$a_n \neq 0$ 時）}$$
相角（諧波）形式
$$f(x) \sim \frac{a_0}{2} + \sum_{n=1}^{\infty}c_n\cos(n\omega_0 x + \delta_n)$$

其中：

$\omega_0 = 2\pi/p$（基本角頻率）

$c_n = \sqrt{a_n^2+b_n^2}$（第 $n$ 階諧波振幅）

$\delta_n = -\arctan(b_n/a_n)$（第 $n$ 階相角）

$\cos(n\omega_0 x + \delta_n)$ 稱為 $f$ 的第 $n$ 階諧波 (harmonic)

振幅頻譜 (Amplitude Spectrum)

怎麼畫：在座標平面上標離散的點，橫軸是頻率，縱軸是振幅。

具體來說，所有點的縱軸都除以 2：

$n = 0$：標點 $(0,\; |c_0|/2)$，其中 $c_0 = a_0$ — 常數項（直流分量）

$n = 1, 2, 3, \dots$：標點 $(n\omega_0,\; c_n/2)$，其中 $c_n = \sqrt{a_n^2+b_n^2}$

（除以 2 是因為相角形式寫的是 $\frac{a_0}{2} + \sum c_n\cos(\cdots)$，常數項本身就帶 $1/2$，為了一致所以全部除 2）

怎麼看：哪個點越高，代表那個頻率對 $f(x)$ 的貢獻越大。

用途：在訊號處理中，一個聲音可以分解成不同頻率的成分。振幅頻譜就是把這些成分的「音量」畫出來。例如鋼琴的 A4 音（440 Hz），頻譜上在 440 Hz 處會有最高的點，然後在 880 Hz、1320 Hz 等倍頻處有較小的點（泛音）。

為什麼要這樣寫？

普通形式 $a_n\cos + b_n\sin$：兩個數（$a_n, b_n$）描述一個頻率。

相角形式 $c_n\cos(\cdot+\delta_n)$：一個振幅 $c_n$ + 一個相位 $\delta_n$，物理意義更直觀。

$c_n$ 告訴你「這個頻率有多強」，$\delta_n$ 告訴你「這個頻率偏移了多少」。

在電機、聲學、訊號處理中，相角形式是標準寫法。

範例 13.11 ─ $f(x)=x^2$, 區間 $0\le x<3$, 週期 $p=3$
完整推導

$p=3$，$\omega_0 = 2\pi/3$。函數只在 $[0,3)$ 上定義，所以用 $\alpha=0$ 在 $[0,3)$ 上積分。

Step 1：算傅立葉係數

$a_0 = \frac{2}{3}\int_0^3 x^2\,dx = \frac{2}{3}\cdot 9 = 6$

$a_n = \frac{2}{3}\int_0^3 x^2\cos\!\left(\frac{2n\pi x}{3}\right)dx$

兩次分部積分，因 $\cos(2n\pi)=1$，$\sin(2n\pi)=0$：

$$a_n = \frac{9}{n^2\pi^2}$$

$b_n = \frac{2}{3}\int_0^3 x^2\sin\!\left(\frac{2n\pi x}{3}\right)dx = -\frac{9}{n\pi}$

Step 2：轉相角形式

$$c_n = \sqrt{a_n^2+b_n^2} = \sqrt{\frac{81}{n^4\pi^4}+\frac{81}{n^2\pi^2}} = \frac{9}{n^2\pi^2}\sqrt{1+n^2\pi^2}$$ $$\delta_n = -\arctan\!\left(\frac{b_n}{a_n}\right) = -\arctan\!\left(\frac{-9/(n\pi)}{9/(n^2\pi^2)}\right) = -\arctan(-n\pi) = \arctan(n\pi)$$
$$\boxed{f(x) \sim 3 + \sum_{n=1}^{\infty}\frac{9}{n^2\pi^2}\sqrt{1+n^2\pi^2}\;\cos\!\left(\frac{2n\pi x}{3}+\arctan(n\pi)\right)}$$

範例 13.12 ─ $f(x)=x$, $-1\le x<1$, 週期 $2$
完整推導

$L=1$，$\omega_0=\pi$。$f$ 為奇函數 → $d_0 = \frac{1}{2}\int_{-1}^{1}x\,dx = 0$。

$d_n$（$n \neq 0$）：

$$d_n = \frac{1}{2}\int_{-1}^{1}x\,e^{-in\pi x}\,dx$$

分部積分，$u=x$，$dv=e^{-in\pi x}\,dx$：

$$= \frac{1}{2}\left[\frac{x\,e^{-in\pi x}}{-in\pi}\right]_{-1}^1 + \frac{1}{2in\pi}\int_{-1}^{1}e^{-in\pi x}\,dx$$ $$= \frac{1}{2}\left[\frac{e^{-in\pi}}{-in\pi} - \frac{-e^{in\pi}}{-in\pi}\right] + \frac{1}{2in\pi}\cdot\frac{e^{-in\pi x}}{-in\pi}\Big|_{-1}^1$$

第二項 $= \frac{1}{2in\pi}\cdot\frac{e^{-in\pi}-e^{in\pi}}{-in\pi} = \frac{-2i\sin(n\pi)}{-2n^2\pi^2} = 0$（因 $\sin(n\pi)=0$）。

第一項 $= \frac{1}{-2in\pi}[e^{-in\pi}+e^{in\pi}] = \frac{-2\cos(n\pi)}{2in\pi} = \frac{-(-1)^n}{in\pi} = \frac{i(-1)^n}{n\pi}$

$$\boxed{d_n = \frac{i(-1)^n}{n\pi}, \quad f(x) \sim \sum_{\substack{n=-\infty \\ n\neq 0}}^{\infty}\frac{i(-1)^n}{n\pi}e^{in\pi x}}$$

轉為實數形式：配對 $n$ 與 $-n$：

$$d_n e^{in\pi x} + d_{-n}e^{-in\pi x} = \frac{i(-1)^n}{n\pi}(e^{in\pi x}-e^{-in\pi x}) = \frac{2(-1)^{n+1}}{n\pi}\sin(n\pi x)$$

得 $f(x) = x$ 在 $[-1,1]$ 的實數傅立葉級數。振幅頻譜：$|d_n| = 1/(n\pi)$。

第 1 題（20%）─ S-L 分類與特徵值

題目：分類 S-L 問題並求特徵值與特徵函數。

$$(e^{2x}y')' + \lambda e^{2x}y = 0, \quad y(0)=y(\pi)=0, \quad \lambda > 1$$
Q1 解答

Step 1：辨認標準形式

與 $(ry')' + (q+\lambda p)y = 0$ 比較：$r(x) = e^{2x}$，$q(x) = 0$，$p(x) = e^{2x}$。

$r(x) = e^{2x} > 0$，$p(x) = e^{2x} > 0$ 在 $[0,\pi]$ 上。邊界條件 $y(0)=0$，$y(\pi)=0$ 為正規邊界條件。

分類：正規 (Regular) S-L 問題。

Step 2：求解

展開 $(e^{2x}y')' = e^{2x}y'' + 2e^{2x}y'$，除以 $e^{2x}$：

$$y'' + 2y' + \lambda y = 0$$

特徵方程 $r^2 + 2r + \lambda = 0$ → $r = -1 \pm \sqrt{1-\lambda}$。

因 $\lambda > 1$，令 $\lambda - 1 = k^2$（$k > 0$），得 $r = -1 \pm ik$。

$$y(x) = e^{-x}(c_1\cos(kx) + c_2\sin(kx))$$

Step 3：代入邊界條件

$y(0) = c_1 = 0$ → $y(x) = c_2 e^{-x}\sin(kx)$。

$y(\pi) = c_2 e^{-\pi}\sin(k\pi) = 0$。取 $c_2 \neq 0$ → $\sin(k\pi) = 0$ → $k = n$（$n = 1,2,3,\dots$）。

$$\boxed{\lambda_n = 1 + n^2, \quad \varphi_n(x) = e^{-x}\sin(nx), \quad n = 1, 2, 3, \dots}$$
第 2 題（20%）─ 特徵函數展開

題目：展開 $f(x)=1-x$（$0 \le x \le 2$）為下列 S-L 問題的特徵函數級數。

$$y'' + \lambda y = 0, \quad y(0)=y(2)=0$$
Q2 解答

Step 1：特徵值與特徵函數

標準問題，$r=1$，$p=1$，$q=0$。由範例 12.1：

$$\lambda_n = \left(\frac{n\pi}{2}\right)^2, \quad \varphi_n(x) = \sin\!\left(\frac{n\pi x}{2}\right), \quad n=1,2,\dots$$

Step 2：求係數

權重函數 $p(x)=1$，用公式 (12.6)：

$$c_n = \frac{\int_0^2 (1-x)\sin(n\pi x/2)\,dx}{\int_0^2 \sin^2(n\pi x/2)\,dx}$$

分母：$\int_0^2 \sin^2(n\pi x/2)\,dx = 1$。

分子：分部積分

$$\int_0^2 (1-x)\sin\!\left(\frac{n\pi x}{2}\right)dx = \left[-(1-x)\frac{2}{n\pi}\cos\!\left(\frac{n\pi x}{2}\right)\right]_0^2 - \int_0^2 \frac{2}{n\pi}\cos\!\left(\frac{n\pi x}{2}\right)dx$$ $$= \frac{2}{n\pi}\cos(n\pi) + \frac{2}{n\pi} - \frac{4}{n^2\pi^2}\sin(n\pi) = \frac{2}{n\pi}((-1)^n + 1)$$
$$\boxed{c_n = \frac{2}{n\pi}((-1)^n + 1), \quad 1-x = \sum_{n=1}^{\infty}\frac{2((-1)^n+1)}{n\pi}\sin\!\left(\frac{n\pi x}{2}\right)}$$
注意

$n$ 為奇數時 $(-1)^n+1=0$，所以只有偶數項存在：$c_{2m} = \frac{4}{2m\pi} = \frac{2}{m\pi}$。

第 3 題（20%）─ 複數傅立葉級數

題目：(a) 求 $f(t)=e^{|t|}$（$-1 < t < 1$）的複數傅立葉級數；(b) 轉換為實數三角形式。

Q3 解答

(a) 複數傅立葉係數

$L=1$，$\omega_0 = \pi$。注意 $e^{|t|}$ 為偶函數。

$$d_n = \frac{1}{2}\int_{-1}^{1}e^{|t|}e^{-in\pi t}\,dt$$

因 $e^{|t|}$ 為偶函數：

$$d_n = \frac{1}{2}\int_{-1}^{1}e^{|t|}(\cos(n\pi t) - i\sin(n\pi t))\,dt = \frac{1}{2}\cdot 2\int_0^1 e^t\cos(n\pi t)\,dt$$

（$\sin$ 項因奇×偶=奇而積分為零）

$$d_n = \int_0^1 e^t\cos(n\pi t)\,dt = \frac{e(-1)^n - 1}{1+n^2\pi^2}$$

$d_0 = \int_0^1 e^t\,dt = e - 1$

$$\boxed{e^{|t|} \sim \sum_{n=-\infty}^{\infty}\frac{e(-1)^n - 1}{1+n^2\pi^2}\,e^{in\pi t}}$$

(b) 轉實數形式

因 $d_n = d_{-n}$（實數且為偶函數），$b_n = 0$，$a_n = 2d_n$（$n \ge 1$），$a_0 = 2d_0$：

$$a_0 = 2(e-1), \quad a_n = \frac{2(e(-1)^n-1)}{1+n^2\pi^2}$$
$$\boxed{e^{|t|} \sim (e-1) + \sum_{n=1}^{\infty}\frac{2(e(-1)^n-1)}{1+n^2\pi^2}\cos(n\pi t)}$$

只有餘弦項（偶函數特性）。

(a) $|x|$ 的傅立葉展開

$|x|$ 為偶函數 → $b_n = 0$。$L = 1$。

$$a_0 = 2\int_0^1 x\,dx = 1$$ $$a_n = 2\int_0^1 x\cos(n\pi x)\,dx = \frac{2((-1)^n - 1)}{n^2\pi^2}$$

$n$ 為偶數時 $a_n = 0$；$n$ 為奇數時 $a_n = -\frac{4}{n^2\pi^2}$。

$$|x| = \frac{1}{2} - \frac{4}{\pi^2}\sum_{m=0}^{\infty}\frac{1}{(2m+1)^2}\cos((2m+1)\pi x)$$
